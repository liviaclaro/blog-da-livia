# 🌸 Blog da Lívia - Maquiagem & Beleza

Um blog moderno e delicado sobre maquiagem com tema em rosa bebê, desenvolvido com HTML, CSS e JavaScript puro.

## ✨ Características

- **Design Rosa Bebê**: Paleta de cores suaves e delicadas
- **Fontes Elegantes**: Playfair Display para títulos, Poppins para conteúdo, Cormorant Garamond para detalhes
- **Responsivo**: Funciona perfeitamente em desktop, tablet e mobile
- **Animações Suaves**: Efeitos de fade-in ao rolar a página
- **Seções Completas**:
  - Hero section com chamada para ação
  - Grid de posts em destaque
  - Seção sobre a maquiadora
  - Newsletter para inscrição
  - Footer com links de contato

## 🚀 Como Usar

1. **Clonar ou baixar o repositório**
   ```bash
   git clone https://github.com/liviaclaro/blog-da-livia.git
   cd blog-da-livia
   ```

2. **Abrir no navegador**
   - Abra o arquivo `index.html` diretamente no seu navegador
   - Ou use um servidor local (recomendado):
     ```bash
     python -m http.server 8000
     # Acesse: http://localhost:8000
     ```

3. **Adicionar imagens**
   - Coloque suas imagens na pasta `images/`
   - Atualize os caminhos no `index.html`:
     - `images/hero.jpg` - Imagem hero principal
     - `images/post1.jpg` - Imagem do primeiro post
     - `images/post2.jpg` - Imagem do segundo post
     - `images/post3.jpg` - Imagem do terceiro post
     - `images/sobre.jpg` - Sua foto

4. **Personalizar conteúdo**
   - Edite o `index.html` para adicionar seus posts
   - Atualize informações de contato (email, WhatsApp, redes sociais)
   - Modifique textos e descrições
